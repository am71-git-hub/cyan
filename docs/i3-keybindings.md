# i3 Keybindings — Cyan

The **Mod key** is the **Search key** (the magnifying glass key, top row).

---

## Windows

| Key | Action |
|-----|--------|
| Mod + Enter | Open new terminal |
| Mod + Q | Close window |
| Mod + Space | Open app launcher (rofi) |
| Mod + F | Fullscreen toggle |
| Mod + Shift + Space | Toggle floating window |

---

## Focus (move between windows)

| Key | Action |
|-----|--------|
| Mod + Arrow keys | Move focus |
| F1 or Back key | Focus previous window |
| F2 or Forward key | Focus next window |

---

## Move Windows

| Key | Action |
|-----|--------|
| Mod + Shift + Arrow keys | Move window |

---

## Workspaces

| Key | Action |
|-----|--------|
| Mod + 1 / 2 / 3 / 4 | Switch to workspace |
| Mod + Shift + 1 / 2 / 3 / 4 | Move window to workspace |

---

## Layout / Splitting

| Key | Action |
|-----|--------|
| Mod + H | Split horizontally (side by side) |
| Mod + V | Split vertically (top and bottom) |

Open a new terminal after splitting to fill the new pane.

---

## Resize Mode

Press **Mod + R** to enter resize mode, then:

| Key | Action |
|-----|--------|
| Arrow keys | Resize window |
| Enter or Escape | Exit resize mode |

---

## Chromebook Top Row Keys

| Key | Action |
|-----|--------|
| F6 | Brightness down |
| F7 | Brightness up |
| F8 | Mute toggle |
| F9 | Volume down |
| F10 | Volume up |
| Print Screen | Screenshot (flameshot) |

---

## i3 System

| Key | Action |
|-----|--------|
| Mod + Shift + R | Reload i3 config |
| Mod + Shift + E | Exit i3 (shows confirmation) |

---

## Bar (polybar)

The bar shows from left to right:

- **Workspace numbers** — click or use Mod+1-4 to switch
- **+** — click to open a new terminal
- **Weather** — current conditions for Etobicoke, updates every 10 min
- **Network** — WiFi signal dots (●●●● = full), click to open network manager
- **Battery** — charge dots + percentage, click for details
- **Date / Time**
